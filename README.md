# 🤖 RAG AI Assistant (HuggingFace + LangChain)

## 📌 Overview

This project is a Retrieval-Augmented Generation (RAG) based AI assistant that allows users to upload documents (PDF, DOCX, TXT) and ask questions based on the content.

## 🚀 Features

* 📂 Upload documents (PDF, DOCX, TXT)
* 🔍 Semantic search using FAISS
* 🤖 AI-powered answers using FLAN-T5 (HuggingFace)
* ⚡ No API key required
* 🎯 Streamlit-based UI

## 🏗️ Tech Stack

* LangChain
* FAISS
* HuggingFace Transformers
* Sentence Transformers
* Streamlit

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
